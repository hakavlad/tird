## `K3. TIME COST`

**Used in:** Actions `2`|`3`|`6`|`7`

**Data type:** `int`

**Valid values:** From `1` to `2^32 - 1` (`4294967295`)

**Default value:** `4`

### Examples


Context: actions `2`|`3`|`6`|`7` with default value:
```
C1. Time cost (default=4):
    I: time cost: 4
```

Context: actions `2`|`3`|`6`|`7`:
```
C1. Time cost (default=4): 1000
    I: time cost: 1,000
```
