## `D1. INPUT FILE PATH`

**Used in:** Actions `2`-`7`

**Data type:** `str`

**Valid values:** Path to an existing, readable file


**Default value:** Not specified


### Examples

```
D1. File to encrypt: secret.zip
    I: path: 'secret.zip'; size: 19,356,207 B (18.5 MiB)
```
